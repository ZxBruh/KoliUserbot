# ⚡ KoliUserbot

Минималистичный, быстрый и мощный Telegram Userbot

---

## 🚀 Возможности

- ⚙️ Модульная система
- 🔄 Авто-обновления через GitHub (ветка main)
- 🧠 Inline кнопки подтверждения
- 📦 Загрузка модулей (.lm / .ml)
- ⚡ Быстрый отклик и лёгкая архитектура
- 🧩 Гибкий конфиг через UI

---

## 💻 Команды

.пинг — Проверка задержки  
.хелп — Список команд  
.рестарт — Перезапуск бота  
.загруз — Загрузить модуль  
.выгруз — Выгрузить модуль  
.модули — Список модулей  
.инфо — Информация о боте  
.эвал — Выполнить Python код  
.эксек — Выполнить async код  
.устан — Установить модуль  
.обнова — Обновление с GitHub  

---

## 📦 Установка

git clone https://github.com/Zxbruh/KoliUserbot  
cd KoliUserbot  
pip install -r requirements.txt  
python main.py  

---

## 🔄 Обновления

- Проверка обновлений идёт с GitHub  
- Используется ветка main  
- Показывается commit hash  
- Есть подтверждение через inline кнопки  

---

## ⚙️ Конфиг

Открыть настройки:

.конфиг  

Пример:

⚙️ Выберите категорию:  
[Встроенные] [Внешние]  

⚙️ Выберите модуль:  
[Название] [Название] [Название]  

---

## 🧠 Загрузка модулей

.lm — загрузка модуля (reply на файл)  
.ml — показать файл модуля  

---

## ⚡ Примечания

- Работает в Termux / VPS / Replit  
- Оптимизирован под мобильные устройства  
- Не требует тяжёлых зависимостей
